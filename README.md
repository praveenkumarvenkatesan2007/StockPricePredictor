# Stock Price Predictor using Machine Learning

## Overview
This project predicts stock prices using a Linear Regression machine learning model.

## Features
- Downloads real stock data using Yahoo Finance.
- Trains a Linear Regression model.
- Predicts the next day's closing stock price.
- Displays Actual vs Predicted prices.
- Calculates R² Score and Mean Squared Error.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yfinance

## How to Run

1. Install the required libraries:

pip install -r requirements.txt

2. Run the program:

python stock_predictor.py
